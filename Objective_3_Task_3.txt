										Objective 3: ALGORITHM

	Task 3: Implement an algorithm for making a simple calculator with all the operators. (+,-,*,/,%)


1. Ask the user to enter Num1 and Num2
2. Ask the user to enter an Operator
3. If the Operator is “+”, then add the two numbers
4. If the Operator is “-”, then subtract the two numbers
5. If the Operator is “*”, then multiply the two numbers
6. If the Operator is “/”, then divide the two numbers
7. The output of each process can be stored in a result variable
8. The result is then displayed for the user
